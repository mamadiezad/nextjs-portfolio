# Next.js Portfolio 🎨

A modern, responsive developer portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Features a clean dark theme, animated skill bars, project showcase, and contact section.

## ✨ Features

- ✅ **Responsive Design** — Looks great on all devices
- ✅ **Dark Theme** — Modern dark UI with purple accents
- ✅ **Animated Skills** — Animated skill progress bars
- ✅ **Project Showcase** — Grid layout with GitHub links
- ✅ **Smooth Scrolling** — Smooth navigation between sections
- ✅ **SEO Optimized** — Meta tags for search engines
- ✅ **Fast Performance** — Next.js static generation

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| **Next.js 14** | React framework |
| **TypeScript** | Type safety |
| **Tailwind CSS** | Styling |
| **React** | UI library |

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for production

```bash
npm run build
npm start
```

## 📁 Project Structure

```
src/
```
